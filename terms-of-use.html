Description of Service

Blurin is an iOS application that provides face masking functionality for photos. The App:

  • Detects faces in photos using Apple’s Vision framework
  • Applies various masking styles (mosaic, blur, pixelation, etc.) to detected faces
  • Allows manual addition of masking regions
  • Exports edited photos with metadata (EXIF/GPS) removal
  • Operates entirely on-device without cloud processing
